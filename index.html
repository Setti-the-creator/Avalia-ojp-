Questao 1: O que é uma publicação web, quais os dois tipos de aplicações existentes e suas principais diferenças.

Essas aplicações podem ser tanto estáticas que são aquelas usadas para sites mais simples ou dinamicas que são utilizadas para sites com uma maior interatividade. Questao 2: O que é um domínio e um DNS.

R: O dominio é um esta guardado apenas um site, já o DNS é uma rede de sites por exemplo se voce pegar um link de um site e colocar na barra de pequisa é o DNS que vai te levar até ele. ref: http://surl.li/ekhtvo; http://surl.li/fcpevv

Questao 3: Explique o que é um servidor compartilhado e um servidor dedicado, evidenciando suas vantagens e desvantagens.

R: O servidor compartilhado é onde vários usuarios dividem o mesmo serividor e os seus recursos sua vantagem é que é mais barato e é facil de ser utilizado porém tem recursos limitados, menor segurança e controle limitado, já o dedicado ele é exlusivo para apenas uma pessoa o que torna ele mais estável e melhor desempenho porém seu custo é muito mais alto e a manutenção é feita pelo próprio usuario. ref: http://surl.li/pfijiy; http://surl.li/uhplzt

Questao 4: O que é um servidor web, explique os dois principais servidores. Como fazer o backup de um projeto de forma compactada. Bem como restaurá-la.

R: Um servidor web é um sistema de computador que hospeda sites e entraga aos usuarios. Temos o Apache HTTP Server e o Nginx. Para fazer o backup e um projeto voce precisa usar o seguinte comando: "tar -czvf projeto_backup.tar.gz /caminho/do/seu/projeto" e para restaurar use o seguinte comando: "tar -xzvf projeto_backup.tar.gz" (todos devem ser executados no seu prompt de comando, "tar" cria o arquivo compactado, -c cria o arquivo, -z compacta ele, -v mostra o progresso no terminal e -f especifica o nome do arquivo de saida.) ref: http://surl.li/zkailw: http://surl.li/vzbnqv: http://surl.li/moawhf

Questao 5: O que é um servidor de banco de dados, explique como fazer o backup e restaurar esse backup. Utilize imagens para ilustrar

R: Um server de banco de dados é um sistema dedicado ao armazenameto e gerenciamento de dados. Para fazer o backup de um banco de dados voce deve usar o seguinte comando no seu promt de comando: " mysqldump -u [usuário] -p [nome_do_banco_de_dados] > backup.sql", já para fazer ua restauração utilize este comando: "mysql -u [usuário] -p [nome_do_banco_de_dados] < backup.sql". ref: http://surl.li/kizuwa; http://surl.li/bpmhfy

Questao 6:O que é um serviço ssh, e como ele permite acessar um servidor web para transferir arquivos de uma máquina local e o servidor

R: O SSH é um protocolo que permite a comunicação de dois dispositivos diferentes de forma segura. Você pode fazer a transferencia de um arquivo para um servidor usando este seguinte codigo em seu promt de comamdo: "scp /caminho/para/o/arquivo usuario@servidor:/caminho/para/salvar", já para uma máquina local use este código:"scp usuario@servidor:/caminho/para/o/arquivo /caminho/para/salvar". ref: http://surl.li/onwfbu; http://surl.li/djzvel

Questao 7: Explique o que é um Linux e cite as principais distribuições usadas como servidor web.

R:O Linux é um sistema operacional de código aberto baseado no kernel Linux. Ele é usado em uma variedade de dispositivos, desde computadores pessoais e servidores até dispositivos móveis e sistemas embarcados. Tem como principais distribuições usadas como servidor web: Ubuntu server, CentOS, Debian, Fedora Server e Arch Linux. ref: https://canaltech.com.br/seguranca/o-que-e-linux: https://www.linux.org/pages/download/

Questao 8:Explique alguns comando básicos do Linux

R: Entre alguns comandos de Linux temos o: "ls" ele lista os arquivos e diretórios, "rm" ele remove arquivos ou diretórios, "mkdir" ele cria um novo diretorio, "touch" cria um novo arquivo vazio ou atualiza a data de modificação de um arquivo existente e "sudo" que permite executar comandos com privilégios de superusuário.
